# 🗓️ SPRINTS.md — План разработки игры Skitter

Всего 4 спринта (по 1 неделе).  
Каждый спринт заканчивается работающей игрой с новыми функциями.

---

## 🟢 СПРИНТ 1 — Скелет и движение (5–11 мая)

**Цель:** Игра запускается, мышь бегает, кот стоит на месте.

### Задачи команды

**Наставник:**
- Создать репозиторий Skitter
- Написать базовый index.html, style.css, game.js (canvas, игровой цикл)
- Настроить GitHub Pages
- Создать ветки для всех участников

**Мурад (физика):**
- Написать функцию moveMouse() с поддержкой WASD и стрелок
- Мышь не выходит за границы экрана

**Дамир (иллюстрации):**
- Нарисовать временные спрайты (кот и мышь — квадраты с ушами)
- Заменить временные квадраты на спрайты в коде

**Остальные:**
- Изучить код, подготовить инструменты

### Результат спринта
- Открывается страница, мышь бегает по полю
- Кот отображается, но не двигается
- Репозиторий настроен, все имеют доступ

---

## 🟡 СПРИНТ 2 — Физика, предметы, уровни (12–18 мая)

**Цель:** Кот бегает за мышью, сыр собирается, появляется новый уровень.

### Задачи команды

**Мурад (физика):**
- Написать moveCat() — кот идёт к мыши по прямой
- Добавить гравитацию и прыжок (если нужно для платформ)
- Настроить столкновения со стенами

**Иван (карты):**
- Создать level1.json и level2.json (разные карты)
- Добавить стартовые позиции мыши и кота

**Миша (достижения):**
- Сделать счётчик очков (сбор сыра = +10 очков)

**Дамир (иллюстрации):**
- Добавить анимацию бега (2–3 кадра) для кота и мыши

**Наставник:**
- Написать функцию сбора предметов (collectItem)
- Интегрировать уровни в игру (загрузка level1.json)

**Остальные:**
- Подготовить ассеты для следующих спринтов

### Результат спринта
- Кот бегает за мышью
- Сыр собирается, очки растут
- При 20 очках открывается level2

## 🔵 СПРИНТ 3 — Полировка, ловушки, звуки (19–25 мая)

**Цель:** Игра становится красивой, появляются ловушки и звуки.

### Задачи команды

**Денис (препятствия):**
- Добавить лужу масла (мышь скользит)
- Добавить шипы (касание → проигрыш)

**Герман (звуки):**
- Добавить звуки: шаги, сбор сыра, мяу кота, победа, проигрыш

**Тимур (пасхалки):**
- Сделать комбинацию i-C-A-T → кот меняет цвет
- Добавить секретную команду в консоль

**Максим (текстуры):**
- Добавить фоновые текстуры для каждого уровня

**Наставник:**
- Интегрировать звуки и ловушки в игровой цикл

### Результат спринта
- Есть звуки на события
- Есть 2 типа ловушек
- Есть 2 пасхалки
- Игра выглядит стильно

## 🔴 СПРИНТ 4 — Релиз, таблица рекордов, ачивки (26 мая – 1 июня)

**Цель:** Игра полностью готова к публикации и презентации.

### Задачи команды

**Миша (достижения):**
- Сделать таблицу рекордов (localStorage)
- Добавить ачивки: «10 собранных сыров», «прошёл 3 уровня»

**Олег и Артём (архитектура и окружение):**
- Добавить декорации на уровни (деревья, ящики, фонари)

**Герман (сюжет):**
- Написать экран победы и поражения
- Добавить подсказки на экран загрузки

**Тимур (пасхалки):**
- Добавить ещё 1–2 пасхалки (по желанию)

**Наставник:**
- Провести финальное тестирование
- Настроить GitHub Pages для финальной версии
- Исправить баги, которые найдут дети

**Все участники:**
- Протестировать игру на разных устройствах
- Написать короткий отчёт «что сделал»

### Результат спринта
- Полностью работающая игра
- Есть таблица рекордов
- Есть экраны победы и поражения
- Игра доступна по ссылке
- Каждый участник знает, что сделал
