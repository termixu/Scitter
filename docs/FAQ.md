Сборник вопросов и ответов, которые помогут новым участникам быстро влиться в проект и избежать типичных ошибок.

---

## 🔧 Вопросы по коду и инструментам

**Вопрос: Не могу открыть index.html в браузере, показывает пустой экран. Что делать?**

Открой консоль разработчика (F12 → вкладка Console). Посмотри на красные ошибки. Чаще всего проблема в неправильном пути к файлу или синтаксической ошибке в game.js. Сообщи об ошибке в чат с скриншотом.

**Вопрос: Игра тормозит, кот дёргается.**

Проверь, не запущен ли игровой цикл дважды (два вызова requestAnimationFrame). Убедись, что в коде нет бесконечных циклов. Уменьши количество объектов на карте.

**Вопрос: Как добавить новый спрайт?**

Положи файл PNG в папку assets/sprites/. В коде используй new Image(), укажи путь src и в draw() отрисуй через ctx.drawImage().

**Вопрос: Почему не работает git push?**

Возможные причины: забыл сделать git add и git commit, не переключился на свою ветку, конфликт с удалённой версией. Сделай git pull перед push.

**Вопрос: Как отменить последний коммит, если случайно запушил?**

Через git reset --soft HEAD~1, затем git push --force. Будь осторожен, эта команда опасна для общей ветки main. В своей ветке — можно.

## 🧠 Вопросы по процессу

**Вопрос: Я сделал задачу, но не знаю, как показать.**

Создай Pull Request из своей ветки в main. Наставник проверит код и либо примет, либо попросит исправить.

**Вопрос: Что делать, если не понимаю задачу?**

Спроси в чате. Напиши, что именно непонятно. Лучше задать вопрос, чем делать не то.

**Вопрос: Какой редактор кода лучше всего подходит?**

VS Code. Он бесплатный, мощный, с хорошими расширениями. GitHub Desktop для тех, кто боится консоли.

**Вопрос: Я новичок, с чего начать?**

Прочитай ONBOARDING.md. Установи VS Code и GitHub Desktop. Склонируй репозиторий. Наставник назначит простую задачу (например, добавить картинку или поправить текст).

## 🎮 Вопросы по игре

**Вопрос: Как добавить новый уровень?**

Создай файл level3.json в папке levels/ по образцу level1.json. Добавь координаты стен, ловушек, стартовую позицию. Оповести Ивана, он интегрирует уровень в игру.

**Вопрос: Как сделать свою пасхалку?**

Открой скрипт scripts/eastereggs.js. В functions добавишь условие и действие. Опиши в комментариях, что делает пасхалка. Тимур поможет.

**Вопрос: Как изменить скорость кота?**

Найди переменную cat.speed в game.js. Измени число. Не меняй без обсуждения с командой — это влияет на баланс игры.

## 🚀 Технические вопросы

**Вопрос: Какие версии браузеров поддерживаются?**

Chrome, Edge, Firefox последних версий. Safari поддерживается, но некоторые эффекты могут работать медленнее.

**Вопрос: Нужен ли сервер для игры?**

Нет, игра полностью клиентская. GitHub Pages хостит статические файлы бесплатно.

**Вопрос: Можно ли играть на телефоне?**

В MVP — только на компьютере. В будущих версиях добавим сенсорное управление.
