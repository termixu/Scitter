## 📌 Примечания

- Если в Pull Request изменены несколько файлов из разных зон, назначаются все указанные кодовладельцы.
- @Наставник является кодовладельцем по умолчанию для всех файлов, не перечисленных выше.
- Для добавления нового кодовладельца используй формат: **@username**

---

## 🧠 Пример чтения записи

**/assets/sprites/** @Дамир

Означает, что Дамир отвечает за все файлы в папке assets/sprites/ и всех её подпапках.
