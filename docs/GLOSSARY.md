# 📖 GLOSSARY.md — Словарь юного разработчика iKODe

**AI (Искусственный интеллект)** — способность компьютера «думать». В нашей игре это кот Nyx, который сам решает, куда идти.

**Assets (ассеты)** — все файлы, которые не код: картинки, звуки, уровни.

**Branch (ветка)** — отдельная версия проекта. Можно экспериментировать, не ломая основной код.

**Bug (баг)** — ошибка в коде. Кот прошёл сквозь стену? Это баг.

**Canvas** — область на веб-странице, где мы рисуем игру.

**Collision (столкновение)** — момент, когда два объекта встретились. Мышь коснулась сыра → сыр исчез.

**Commit (коммит)** — сохранение изменений в Git. Как чекпоинт в игре.

**Console (консоль)** — инструмент разработчика (F12). Там видны ошибки.

**Ctrl+Z** — отмена последнего действия. Работает везде.

**Debug (отладка)** — поиск и исправление ошибок.

**DOM** — представление веб-страницы в виде дерева объектов.

**Flexbox** — способ раскладывать блоки в ряд или колонку.

**FPS** — кадры в секунду. 60 — идеально для игры.

**Git** — программа, которая отслеживает изменения в коде.

**GitHub** — сайт, где хранятся Git-репозитории.

**Hitbox** — невидимый прямоугольник вокруг персонажа для столкновений.

**Hover** — состояние, когда мышь навели на элемент.

**IDE** — программа для кода. Наша — VS Code.

**If** — условие: «если верно — сделай это».

**Issue** — задача на GitHub («добавить звук», «исправить баг»).

**JSON** — формат для хранения данных. Используем для уровней.

**Keyframes** — правила для CSS-анимации.

**LocalStorage** — хранилище в браузере. Там лежат рекорды.

**Loop (цикл)** — повторение кода много раз.

**Main** — основная ветка проекта с работающим кодом.

**Merge (слияние)** — объединение изменений из разных веток.

**Pixel art** — рисование квадратиками-пикселями.

**PNG** — формат картинок с прозрачным фоном.

**Pull Request** — запрос на проверку твоего кода.

**Push** — отправка твоих изменений на GitHub.

**Render (рендер)** — отрисовка кадра.

**Repository (репозиторий)** — папка с проектом на GitHub.

**RequestAnimationFrame** — запуск следующего кадра. Основа игрового цикла.

**Scale** — изменение размера.

**Script (скрипт)** — программа (код).

**Sprite (спрайт)** — картинка персонажа или предмета.

**Sprint (спринт)** — неделя работы над задачами.

**Transition** — плавное изменение стилей.

**Variable (переменная)** — коробка с именем для хранения данных.

**Wrap** — перенос элементов на новую строку, если не влезают.
