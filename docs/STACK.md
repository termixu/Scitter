# 🛠️ STACK.md — Технологический стек проекта Skitter

Здесь описан набор инструментов, языков и сервисов, которые мы используем для разработки игры.

---

## 🎮 Основные технологии

**HTML5**  
Структура страницы. Создаёт canvas, кнопки, экраны.

**CSS3**  
Оформление. Цвета, шрифты, расположение элементов, адаптация под экран.

**JavaScript (ES6+)**  
Вся логика игры: движение, столкновения, очки, уровни, звуки.

**Canvas API**  
Рисование 2D-графики. Отрисовка персонажей, предметов, фона.

---

## 🧰 Инструменты разработки

**VS Code**  
Основной редактор кода.

**Расширения VS Code**

- Live Server — для быстрого просмотра изменений
- Prettier — автоформатирование кода
- ESLint — подсветка ошибок

**GitHub Desktop**  
Управление репозиторием без консоли (для детей, кто боится командной строки)

**Git**  
Система контроля версий (через консоль или GitHub Desktop)

**Браузер (Chrome / Edge)**  
Запуск и отладка игры. Инструменты разработчика (F12) для поиска ошибок.

---

## 🎨 Графика и анимация

**Piskel**  
8bit-спрайты. Онлайн, бесплатно, без установки.

**GIMP**  
Редактирование текстур, создание фонов.

**Aseprite** (опционально)  
Профессиональный инструмент для пиксель-арта. Платная версия.

---

## 🔊 Звук и музыка

**Audacity**  
Запись и редактирование звуков.

**FreeSound.org**  
Бесплатные звуковые эффекты (с указанием авторства).

**BFXR**  
Генератор ретро-звуков для игр (пиксельные звуки, взрывы, прыжки).

---

## 🗺️ Уровни и данные

**JSON**  
Формат хранения уровней (level1.json, level2.json).

**Tiled** (опционально)  
Визуальный редактор карт. Экспорт в JSON.

## 📋 Система контроля версий (Git)

Основные команды (для детей, кто работает в консоли):

- `git status` — проверить изменения
- `git add .` — добавить все файлы под сохранение
- `git commit -m "текст"` — сохранить изменения
- `git push` — отправить на GitHub
- `git pull` — забрать изменения с GitHub
- `git branch` — показать ветки
- `git checkout -b имя` — создать ветку и переключиться
